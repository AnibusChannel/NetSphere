Документация NetSphere

Введение

Данная система предназначена для управления устройствами в корпоративной сети компании. Сеть организована в виде иерархии доменов, содержащих различные типы устройств.

Архитектура

Система состоит из следующих основных компонентов:

  • Модуль хранилищ данных
  • Модуль тестирования

Проекты решения

Решение состоит из двух проектов:

  • NetSphere - основное приложение
  • NetSphereTests - модуль тестирования

Использование

Для работы с системой необходимо создать экземпляр CorporateNetwork и добавлять к нему устройства и домены.

Разработка

Система разрабатывается на C++ с использованием Google Test для модульного тестирования.
