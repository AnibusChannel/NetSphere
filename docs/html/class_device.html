Подробное описание

Абстрактный базовый класс для всех устройств корпоративной сети.

Наследует от NetworkEntity и добавляет общие свойства устройств: MAC-адрес и методы для работы с ним.

Проверяет валидность MAC-адреса.

Аргументы
[in]macMAC-адрес для проверки.
Возвращает
true если MAC-адрес валиден, иначе false.

Валидный MAC-адрес должен соответствовать формату: XX:XX:XX:XX:XX:XX или XX-XX-XX-XX-XX-XX, где X - шестнадцатеричная цифра.
